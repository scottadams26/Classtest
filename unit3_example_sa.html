Overview¶

Problem Statement¶

Analyzing credit trends is important for gaining a better understanding of the financial security of a population. Indeed, possessing a good credit score provides access to financing for major investments such as cars, housing, and education. However, a credit profile takes time to build and a short credit history is one factor that can have a negative impact on an individuals credit score.

Proposed Test¶

This analysis tested the hypothesis that credit scores are, on average, higher among older individuals.

Measurable Outcome¶

Using the Churn-Modelling data, I examined whether age is associated with credit score. More specifically, I tested the hypothesis that being older will be associated with higher credit scores, on average.

If data are missing completely at random (MCAR), meaning that an observation's probability of having missing data on a given variable does not depend on the true value of the variable in question or the values of any other variables. In this case, dropping observations with missing data on any variables that will be used in the analyses would result in a loss of data, but would not bias the data. The uncommented syntax below will drop observations with any missing data on Age or CreditScore. The commented code shows an alternative approach, which is to set missing values on a variable to the variable's mean. Though this approach is used by some, I recommend not using it for a number of reasons, including the fact that imputing missing data with the variable's mean artifically minimizes the measure's variance. Additionally, the mean that is imputed on to missing observations is the variable's mean excluding missing data! Thus, if observations with missing data on the variable in question have systematically different values on the variable in question compared to observations with valid data on said variable, bias will be introducted.

The plotted regression line better illustrates the weak negative relationship between age and credit score. In addition, the $R^2$ value is less than 0.000, further demonstrating that this model is not performing well. Part of the inability of the analyses to detect a meaningful relationship between age and credit score may be due to the fact that the data are quite sparse at older ages. Therefore, instead of treating age as a continuous variable, I recoded age into three mutually exclusive categories:

  • Young Adult: Age 18-29,
  • Middle Age: Age 30-64,
  • Older Adult: Age 65+.

Below, the where function from the Numpy library is used to create three mutually exclusive variables, also referred to as "dummy variables." np.where is just a conditional statement where the first argument in parentheses provided the condition to be evaluated, the second argument is the value to be returned if the condition is True, and the third argument is the value to be returned if the condition is False. For example, churn['age18_29'] = np.where(churn['Age'].between(18, 29), 1, 0) creates a new column in the churn dataframe called age18_29 which takes a value of 1 when Age is between 18 and 29, and a value of 0 otherwise. Since these new age ategories are coded as 0 and 1, the mean of each age category will be the proportion of cases coded as a 1 on the variable, and the sum of all three proportion will equal 1.

So, why go through all this trouble, and code, to run a linear regression model with dummy variables when you could get predicted values by examining the mean outcome by each group? Great question! In the case of a simple linear regression model with one predictor (or in this case one set of dummy variables), there is little benefit to conducting a linear regression to get predicted values. However, the linear regression model does providethe $R^2$ value, which is a nice metric of model fit. In this example, the mean credit score is higher for older adults, but the mean credit score among older adults is only about 10 points higher than either of the younger age groups. Taking this result alongside the small $R^2$ provides a stronger case that age may not be the best predictor of credit scores.

Conclusion¶

The linear regression results using age groups as predictors of credit score, as opposed to the continuous age value, is consistent with the initial hypothesis that older individuals tend to have better credit scores than younger individuals. However, the $R^2$ value for the model with the dummy variable predictors was still very low, indicating that age is not a very strong predictor of credit score. Thus, the relationship between age and credit score is not a very meaningful relationship based on the Churn-Modeling data.
